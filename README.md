# license-plate-detection-yolov8
I recently completed the next step of my License Plate Detection project.

Earlier, I worked on creating a custom dataset using Roboflow, where I collected image data, annotated license plates, organized the dataset, and exported it in YOLO format. After preparing the dataset from scratch, I have now trained the model using YOLO / Ultralytics and successfully tested it on video footage.

This project helped me understand the full workflow of an object detection project, from dataset preparation to model training and real-time video detection.

Current progress:
1. Custom dataset creation using Roboflow  
2. License plate annotation and YOLO format export  
3. YOLO model training using Ultralytics  
4. License plate detection on video  
5. Custom bounding box output generation  

This was a great learning experience for me because I built the workflow step by step, starting from basic dataset preparation and moving toward model training and video-based detection.

My next version will focus on OCR integration, where the detected license plate will be cropped, read automatically, and displayed clearly in a larger view on the video output.

I am excited to continue improving this project and learning more about Computer Vision, Object Detection, OCR, and AI-based automation.
